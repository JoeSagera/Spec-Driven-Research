# SDR Framework - Code Review Standards

## Documentation
- Use Markdown with YAML frontmatter for skill files
- Follow existing SDD patterns from Gentleman Programming
- Include decision gates and result contracts

## Structure
- Organize skills in dedicated directories (skills/sdr-<phase>/SKILL.md)
- Agent templates in skills/agents/
- Documentation files at skills/ root level

## Style
- Consistent YAML frontmatter format
- Professional, actionable language
- Include Engram persistence instructions
